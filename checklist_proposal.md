# 스마트 정기검사 체크리스트 (Smart Inspection Guide) 제안

다운로드하신 PDF(`방사선분야 정기검사수검가이드(2022년 개정)_최종본.pdf`)의 목차와 구조를 분석하여, 스마트폰 환경에 최적화된 **3단계 체크리스트 구조**를 제안합니다.

## 1. 구조 설계 (Structure)
> 사용자가 "어디를 점검해야 하는지" 직관적으로 알 수 있도록 **3단계 깊이(Depth)**로 구성합니다.

- **Level 1 (카테고리/탭)**: 대분류 (예: 서류, 시설, 인력...)
- **Level 2 (체크 항목)**: 구체적인 점검 대상 (아코디언 헤더)
- **Level 3 (세부 가이드)**: 실행 지침 및 확인 방법 (아코디언 바디)

## 2. 데이터 모델 (Data Model)
각 항목은 아래와 같은 속성을 가집니다.

```typescript
interface InspectionItem {
  id: string;        // 고유 ID (예: "doc-01")
  category: string;  // 대분류 (예: "서류", "시설")
  title: string;     // 항목명 (예: "과년도 지적사항 조치")
  description: string; // 세부 설명 (마크다운 지원)
  importance: '필수' | '권장'; // 중요도
  status: 'pending' | 'ok' | 'fail'; // 사용자가 체크한 상태 (로컬 저장)
}
```
